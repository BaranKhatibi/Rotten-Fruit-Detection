# Fruit Freshness Classification using SVM

This repository contains one of my earliest Python machine learning projects, created around 5 years ago (2021). The code is not optimized and the results are not particularly strong, but I’m keeping it here as part of my learning history.  If I were rebuilding it today, I would approach the problem differently and improve it today. Still, it represents an important step in my journey.

The project uses classical computer vision and an SVM classifier to distinguish between fresh and rotten fruits.

The implementation is not optimized and reflects my early learning stage, but I’m sharing it as part of my development journey.
